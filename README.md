## Getting Started

### Prerequisites:

There are several required Python packages to run the code:
- gensim (for node2vec code)
- numpy
- Scikit-learn
- imblearn
- pandas
- xgboost

These packages can be installed using pip or conda as the follwoing example
```
pip install -r requirements.txt
```
